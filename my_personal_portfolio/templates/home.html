Vehicle Number Plate Recognition System

TechStack : Coding in Python, Tesseract OCR, OpenCV, Image Processing

OCR stands for Optical Character Recognition
Optical character recognition or optical character reader (OCR) is the electronic or mechanical conversion of images. We have used this technique to read the number plates of the vehilces.

The tesseract library is an optical character recognition (OCR) tool for Python. That is, it can recognize and read the text embedded from any image. So we’ll use it for identifying the characters inside the number plate.

Using Image processing, the ROI of the image i.e The number plate is extracted nad then using Tesseract OCR, the contents of plate are extracted and stored in a database.
